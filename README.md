# Weather

A simple CLI to know whether or not it is going to rain today.

## Usage

```
$ python main.py NAME
```

Example

```
$ python main.py Paris
It is going to rain today in Paris.
```

## Installation

```
$ pip3 install requirements.txt
$ py.test -v
```
